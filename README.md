# chatgptapp

# react , node

# Backend

cd server <br>
yarn <br>
Create a .env file in the server folder <br>
yarn start <br>

# Frontend

cd client <br>
yarn <br>
Create a .env.local file in the client folder(for local development) <br>
Create a .env.production file in the client folder(for production) <br>
yarn dev <br>
